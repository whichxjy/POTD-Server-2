# POTD-Serve 2

[POTD-Server](https://github.com/whichxjy/POTD-Server) 2.0, powered by Apache Thrift.

## Dependency

* Apache Thrift

## Usage

* Server

    1. Change the directory to `server`.

    2. Run the program: `make run`.

* Client

    1. Change the directory to `client`.

    2. Run the program: `make run`.
